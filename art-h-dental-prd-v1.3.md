## 1. 프로젝트 개요

송도국제업무단지 C8-2블럭에 개업하는 아트에이치치과의 공식 홈페이지.
아임웹 스타일의 감성적 디자인을 커스텀 코드로 직접 구현하며, 멀티페이지 구조로 제작한다.

### 핵심 결정사항

| 항목 | 결정 |
|------|------|
| 플랫폼 | 직접 커스텀 코드 구현 (아임웹 스타일) |
| 구조 | 멀티페이지 (6개 독립 페이지) |
| 배경 | 흰색 바탕 클린 디자인 |
| 모션 | 전 요소 스크롤 기반, 요소별 속도 차별화 |
| 이미지 | 개원 전 CSS 비주얼 플레이스홀더 → 개원 후 실사 교체 |
| AI 방지 | 비대칭 레이아웃, 서체 차별화, 골드 절제, 감성 카피 |

## 2. 디자인 시스템 (v1.3 확정)

### 2.1 AI 느낌 제거를 위한 6대 개선 원칙

| 문제 | 개선 | 적용 |
|------|------|------|
| 반복 섹션 구조 | 섹션마다 다른 레이아웃 | 풀블리드/비대칭/메이슨리/수평 등 혼합 |
| Cormorant 클리셰 | DM Serif Display로 교체 | 덜 뻔하면서 격조 유지 |
| 골드 남발 | 핵심 CTA/야간강조에만 한정 | 나머지는 차콜+밑줄 링크 |
| 균일 모션 | 요소별 속도/종류 차별화 | 제목 0.6s, 이미지 1.2s, 본문 0.8s |
| 대칭 공간 | 5:4, 2:3, 풀블리드 비대칭 | 50:50 그리드 지양 |
| AI 카피 | 환자 시점 감성 문장 | 수식어 나열 대신 구체적 이야기 |

### 2.2 서체

| 용도 | 서체 | 비고 |
|------|------|------|
| 영문 Display | DM Serif Display | Cormorant 대체, AI 공식 회피 |
| 한글 본문 | Pretendard Variable | 300~600 가변 |
| 소라벨 | Pretendard 12px | letter-spacing: 3px, 회색 |

### 2.3 컬러

| 용도 | 컬러 | 비고 |
|------|------|------|
| 배경 | #FDFCFA | 순백이 아닌 웜 화이트 |
| 보조 배경 | #F5F2ED | 웜 베이지 |
| 텍스트 | #2E2E2E / #6B6B6B | 순흑 아닌 차콜 |
| 골드 | #A68B5B | CTA, 야간진료 강조에만 |
| 골드 라이트 | #C4AA7A | 히어로 서브텍스트에만 |
| 구분선 | rgba(0,0,0,0.07) | 극도로 연함 |

### 2.4 모션 위계

| 요소 | 속도 | 이징 | 방향 |
|------|------|------|------|
| 제목 | 0.6~0.7s | cubic-bezier(.16,1,.3,1) | translateY |
| 이미지 | 1.0~1.3s | 동일 | scale / translateX |
| 본문 | 0.8~0.9s | 동일 | translateY |
| 소요소 | 0.5s | 동일 | translateY |
| 히어로 | 0.6~1.2s | ease | 순차 딜레이 0.4~1.3s |
| 패럴랙스 | scroll * 0.1 | — | translateY |

### 2.5 이미지 전략

데모/개발 환경에서는 CSS 그라데이션 기반 비주얼 플레이스홀더 사용.
각 영역의 분위기에 맞는 색감을 적용하여 완성본에 가까운 시각적 인상을 제공.

| 영역 | 색감 | 교체 예정 이미지 |
|------|------|-----------------|
| 히어로 | 다크 네이비 (#2C3E50→#1a1a2e) | 진료실/의원 대표 영상 |
| 의원 내부 | 웜 베이지 (#e8e0d4→#c9b896) | 대기실 전경 사진 |
| 진료실 | 쿨 블루그레이 (#dce3e8→#a3b5c4) | 진료 체어 사진 |
| 원장님 | 뉴트럴 (#e0ddd8→#b0a999) | 원장님 프로필 사진 |
| 임플란트 | 쿨 블루 (#d6dee4→#8ea8bc) | 임플란트 시술 장면 |
| 대기실 | 웜 크림 (#ece8e0→#d0c4b0) | 대기실 인테리어 |
| 수술실 | 쿨 그레이 (#dce0e4→#a8b4c0) | 독립 수술실 사진 |
| 장비 | 스틸 블루 (#dee2e6→#98aab8) | CT/CEREC 장비 사진 |
| 도시 | 다크 블루 (#2c3e50→#1a2530) | 송도 야경/도시 전경 |

## 3. 사이트맵

```
Art H Dental
├── Home
│   ├── 풀블리드 히어로 (좌측 하단 정렬)
│   ├── 브랜드 소개 (5:4 비대칭)
│   ├── 진료과목 3종 프리뷰 (임플란트, 신경치료, 사랑니 발치)
│   ├── 의료진 (풀블리드 50:50 분할)
│   ├── 시설 (2:1 메이슨리 갤러리)
│   └── 위치 (수평 미니멀)
├── 의원소개
│   ├── 페이지 히어로 (좌측 하단 타이틀)
│   ├── 브랜드 스토리 (좌정렬 산문)
│   ├── 진료 시스템 (수평 리스트)
│   └── 풀블리드 이미지
├── 의료진
│   ├── 페이지 히어로
│   └── 프로필 (좌 스티키 사진 / 우 경력)
├── 진료과목
│   ├── 페이지 히어로
│   └── 7개 항목 (좌 이미지 / 우 텍스트 가로카드)
├── 시설 · 장비
│   ├── 페이지 히어로
│   ├── 진료 공간 (2x2 이미지 그리드)
│   └── 보유 장비 (이미지 + 텍스트 수평 리스트)
└── 오시는길
    ├── 페이지 히어로
    └── 주소/연락처/진료시간/지도 (5:4 2컬럼)
```

## 4. 기술 스택

| 구분 | 기술 |
|------|------|
| Frontend | Next.js 14+ App Router |
| 서체 | Google Fonts (DM Serif Display) + Pretendard CDN |
| 애니메이션 | Intersection Observer + CSS Transitions |
| 이미지 | CSS Gradient Placeholders → 실사 교체 |
| 배포 | Vercel |
| 지도 | 카카오맵 / 네이버맵 API |
| 예약 | 네이버 예약 외부 링크 |

- "진료 너머, 사람의 고귀함을 생각합니다" (인본주의적 슬로건)
